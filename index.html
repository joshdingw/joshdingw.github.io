Quant Knowledge Graph

A graph-based learning prototype for quantitative research, factor investing, and systematic learning.

Why this site?

Most online learning resources are fragmented, linear, and difficult to reuse. This project explores a new way to organize quant knowledge through linked nodes, learning paths, and reusable modules.

Core Modules

  • Factor Models
  • Time Series
  • Backtesting
  • Portfolio Construction
  • Machine Learning for Alpha
  • Market Microstructure

Current Goal

I am using this project to systematize my own quant learning, build a public knowledge graph, and gradually develop it into a collaborative learning platform.
